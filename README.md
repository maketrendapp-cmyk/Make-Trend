## 📖 About

**Make Trend** is a modern platform that lets creators, marketers, and entrepreneurs launch **share-to-unlock viral campaigns** in under 2 minutes. Choose from professionally designed templates, customize your campaign, share your unique link, and track real-time metrics like views, shares, unlocks, and completions.

This repository contains the **frontend** – a fast, responsive, and PWA-ready Next.js application.

> 🔗 **Backend Repository:** [Make Trend Backend](https://github.com/maketrendapp-cmyk/Make-Trend/tree/main/backend)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Template Library** | Browse and use professionally designed campaign templates |
| ⚡ **Campaign Creation** | Create campaigns with share counts, tasks, and redirect URLs |
| 📊 **Real-Time Analytics** | Track views, shares, unlocks, completions, and campaign success rate |
| 🔐 **Authentication** | Email/Password + Social Login (Google, Facebook) |
| 📱 **PWA Ready** | Install as a Progressive Web App on mobile and desktop |
| 👥 **Referral System** | Invite friends and earn rewards with unique referral codes |
| 💬 **Community Reviews** | Users can leave reviews and ratings |
| 🎯 **Infinite Scroll** | Smooth pagination for campaign lists |
| 📦 **Smart Caching** | React Query for efficient data fetching and cache management |

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| **Framework** | [Next.js 15](https://nextjs.org/) (Pages Router) |
| **UI Library** | [React 18](https://reactjs.org/) |
| **Styling** | [Tailwind CSS 3](https://tailwindcss.com/) + Framer Motion |
| **State Management** | [React Query (TanStack Query)](https://tanstack.com/query) |
| **Authentication** | [Firebase Auth](https://firebase.google.com/docs/auth) |
| **Database** | [Firebase Firestore](https://firebase.google.com/docs/firestore) (via backend) |
| **Icons** | React Icons (Fi, Fa) |
| **Animations** | Framer Motion |
| **PWA** | next-pwa |
| **Deployment** | [Vercel](https://vercel.com) |
